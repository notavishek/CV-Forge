Copy your SwiftLaTeX WASM build output into this folder.

Expected files referenced by index.html (XeTeX preferred for fontspec):
- swiftlatex/XeTeXEngine.js
- swiftlatex/swiftlatexxetex.js
- swiftlatex/swiftlatexxetex.wasm

Optional fallback (PdfTeX):
- swiftlatex/PdfTeXEngine.js
- swiftlatex/pdftex.wasm

If your build outputs different names, update index.html accordingly.
