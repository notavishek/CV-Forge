CV Forge (SwiftLaTeX WASM)

This is a browser-only CV builder that collects structured CV data, renders LaTeX, compiles with SwiftLaTeX WASM, and exports a PDF.

Setup
- Copy your SwiftLaTeX WASM build into the swiftlatex folder (XeTeXEngine.js + swiftlatexxetex.js + swiftlatexxetex.wasm).
- Add the font files listed in assets/README.txt.
- Serve this folder with a local static server and open the page in your browser.

Notes
- Compilation happens in-browser and may take a few seconds.
- Replace placeholder data in the form with your real CV details.
